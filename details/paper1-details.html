Main Idea

The capacity of a discrete memoryless channel is given by:

\[ C = \max_{p(x)} I(X;Y) \tag{1} \label{eq:capacity} \]

Equation \(\ref{eq:capacity}\) shows that channel capacity is determined by maximizing mutual information (see Fig.~\ref{fig:main}, also discussed in [\cite{ref1}]).

Results

Using random coding arguments, one can show:

\[ \lim_{n \to \infty} P_e^{(n)} = 0 \tag{2} \label{eq:error} \]

Equation \(\ref{eq:error}\) indicates that as the blocklength grows, error probability vanishes. Numerical values are summarized in Table~\ref{tab:results} (see also [\cite{ref2}]).
